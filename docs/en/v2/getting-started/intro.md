## Folder structure

All docs live under the repository root `docs/` folder, split by language and version:

- `docs/zh/v2/...`
- `docs/en/v2/...`

## Routing

The page URL follows the file path:

- `docs/en/v2/getting-started/intro.md` → `/en/docs/v2/getting-started/intro`
