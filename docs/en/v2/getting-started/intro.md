## What is ElexvxAI?

At its core, ElexvxAI provides a unified interface for interacting with various generative AI models. Whether you are building a chatbot, a content generation tool, or an automated analysis pipeline, our SDK simplifies the complexity of model management.

## Installation

To get started, install the package via npm or yarn. Ensure you have Node.js version 18 or higher installed.

```bash
$ npm install @elexvx-ai/core
$ npm install @elexvx-ai/react-components
```
