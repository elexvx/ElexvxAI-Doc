# 快速入门教程

本教程将带您快速了解 ElexvxAI Lab 的核心功能，并构建您的第一个 AI 应用。

## 前置条件

在开始之前，请确保您已经：

- ✅ 完成了[安装指南](/docs/getting-started/installation)
- ✅ 获取了 API 密钥
- ✅ 配置了环境变量

## 第一个 AI 聊天应用

让我们创建一个简单的 AI 聊天应用。

### Step 1: 创建新项目

```bash
elexvx-lab create chat-app
cd chat-app
```

### Step 2: 创建聊天组件

创建 `src/chat.js` 文件：

```javascript
import { ElexvxAI } from '@elexvxai/lab';

// 初始化客户端
const client = new ElexvxAI({
  apiKey: process.env.ELEXVX_API_KEY,
});

async function chat(message) {
  const response = await client.chat.completions.create({
    model: 'gpt-4',
    messages: [{ role: 'user', content: message }],
    temperature: 0.7,
    max_tokens: 500,
  });

  return response.choices[0].message.content;
}

export { chat };
```

### Step 3: 运行应用

```bash
node src/index.js
```

## 核心概念

### 1. 客户端初始化

```javascript
const client = new ElexvxAI({
  apiKey: process.env.ELEXVX_API_KEY,
  baseURL: 'https://api.elexvxai.com',
  timeout: 30000,
});
```

### 2. 模型选择

支持多种模型：

- `gpt-4` - 最强大的模型，适合复杂任务
- `gpt-3.5-turbo` - 快速且经济，适合日常对话
- `claude-3` - 擅长长文本处理
- `llama-2` - 开源模型，支持本地部署

### 3. 消息格式

对话由消息数组组成：

```javascript
const messages = [
  { role: 'system', content: '你是一个有帮助的助手' },
  { role: 'user', content: '用户的问题' },
  { role: 'assistant', content: 'AI 的回答' },
];
```

## 实用示例

### 文本摘要

```javascript
async function summarize(text) {
  const response = await client.chat.completions.create({
    model: 'gpt-3.5-turbo',
    messages: [
      { role: 'system', content: '请将以下文本总结为 3-5 句话' },
      { role: 'user', content: text },
    ],
  });
  return response.choices[0].message.content;
}
```

### 情感分析

```javascript
async function analyzeSentiment(text) {
  const response = await client.chat.completions.create({

```

## 最佳实践

### 错误处理

始终使用 try-catch 处理 API 调用：

```javascript
try {
  const response = await client.chat.completions.create({...});
} catch (error) {
  if (error.status === 429) {
    console.log('请求过于频繁，请稍后重试');
  } else if (error.status === 401) {
    console.log('API 密钥无效');
  } else {
    console.log('发生错误:', error.message);
  }
}
```

### 控制成本

使用 `max_tokens` 限制响应长度：

```javascript
const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: messages,
  max_tokens: 500, // 限制最大令牌数
});
```
