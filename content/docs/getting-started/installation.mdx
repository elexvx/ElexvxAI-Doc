# 安装指南

本指南将带您完成 ElexvxAI Lab 的安装和配置过程。

## 系统要求

在开始之前，请确保您的系统满足以下要求：

### 硬件要求

- **CPU**: 4 核心或更多（推荐 8 核心）
- **内存**: 至少 8GB RAM（推荐 16GB 或更多）
- **存储**: 至少 20GB 可用空间
- **GPU**: （可选）NVIDIA GPU 用于加速推理

### 软件要求

- **操作系统**: Windows 10/11, macOS 12.0+, Linux (Ubuntu 20.04+)
- **Node.js**: 18.0 或更高版本
- **Python**: 3.9 或更高版本
- **Git**: 任意最新版本

## 安装步骤

### 1. 安装 Node.js

从 [Node.js 官网](https://nodejs.org/) 下载并安装 LTS 版本。

验证安装：

```bash
node --version
npm --version
```

### 2. 安装 Python

确保安装 Python 3.9 或更高版本，访问 [Python 官网](https://www.python.org/downloads/)。

检查 Python 版本：

```bash
python --version
```

### 3. 安装 ElexvxAI Lab CLI

使用 npm 全局安装 CLI 工具：

```bash
npm install -g @elexvxai/lab-cli
```

验证安装：

```bash
elexvx-lab --version
```

### 4. 初始化项目

创建新项目：

```bash
mkdir my-ai-project
cd my-ai-project
elexvx-lab init
```

### 5. 配置环境变量

复制示例配置文件：

```bash
cp .env.example .env
```

编辑 `.env` 文件，添加您的配置：

```env
# API 配置
ELEXVX_API_KEY=your_api_key_here
ELEXVX_API_URL=https://api.elexvxai.com

# 模型配置
DEFAULT_MODEL=gpt-4
MAX_TOKENS=2048
```

> **提示**: 您可以从 [开发者控制台](https://console.elexvxai.com) 获取 API 密钥。

## 验证安装

运行以下命令验证一切正常：

```bash
npm run dev
```

访问 `http://localhost:3000` 查看您的应用！
