## 账户与认证

### 如何获取 API 密钥？

1. 访问 [开发者控制台](https://console.elexvxai.com)
2. 登录或注册账户
3. 进入 "API 密钥" 页面
4. 点击 "创建新密钥"
5. 复制并安全保存您的密钥

> **警告**: API 密钥仅在创建时显示一次，请妥善保管。

### API 密钥丢失了怎么办？

如果您的 API 密钥丢失：

1. 登录开发者控制台
2. 撤销旧密钥
3. 创建新密钥
4. 更新应用中的环境变量

### 可以在客户端直接使用 API 密钥吗？

**不推荐！** API 密钥应该保存在服务器端。推荐做法是在后端创建 API 路由。

## 技术问题

### 为什么 API 调用返回 401 错误？

401 错误通常表示认证失败。检查以下几点：

- API 密钥是否正确设置
- 环境变量是否正确加载
- 密钥是否已过期或被撤销

### 为什么会出现 429 错误（请求过多）？

429 错误表示您已达到速率限制。解决方法：

1. 实施重试逻辑
2. 升级套餐以提高速率限制
3. 实施请求缓存

- **Claude 3** - 长文本理解
- **Llama 2** - 开源部署
- **Gemini Pro** - 多模态任务

## 成本与计费

### 如何计算令牌使用量？

令牌是 AI 模型处理文本的基本单位：

- 1 个令牌 ≈ 4 个英文字符
- 1 个汉字 ≈ 1-2 个令牌

使用量 = 输入令牌 + 输出令牌

### 如何优化成本？

降低成本的技巧：

1. 选择合适的模型（简单任务使用 GPT-3.5）
2. 限制输出长度 (`max_tokens`)
3. 缓存响应
4. 精简提示词
5. 批量处理
